# Keyboard Shortcuts

## General

| Shortcut | Action |
|----------|--------|
| ++ctrl+s++ | Save workflow |
| ++ctrl+o++ | Open workflow |
| ++ctrl+z++ | Undo |
| ++ctrl+shift+z++ | Redo |
| ++ctrl+w++ | Run Graph |
| ++ctrl+b++ | Batch Run |
| ++ctrl+f++ | Focus Node Explorer search |
| ++f1++ | Open Manual |
| ++ctrl+q++ | Quit |

## Canvas

| Shortcut | Action |
|----------|--------|
| ++tab++ | Open inline node search (canvas) |
| ++delete++ / ++backspace++ | Delete selected nodes |
| ++ctrl+a++ | Select all nodes |
| ++ctrl+c++ | Copy selected nodes |
| ++ctrl+v++ | Paste nodes |
| ++ctrl+d++ | Duplicate selected nodes |
| Scroll wheel | Zoom in/out |
| Middle-click drag | Pan canvas |
| ++alt++ + drag | Pan canvas |
| ++f++ | Fit all nodes in view |

## Nodes

| Shortcut | Action |
|----------|--------|
| Double-click node | Open preview / detail view |
| Right-click node | Context menu (disable, delete, etc.) |
| Drag from port | Create connection |
| Click connection | Select connection |
| ++delete++ on connection | Remove connection |
